◆ Fund assumptions

Enter both as decimals (0.06 = 6%, 0.25 = 25%).

Hurdle rate is the minimum return Class B earns before any profit is shared. In a year the fund returns more than the hurdle, Class B first receives its hurdle amount off the top. Only the profit above the hurdle is then split.

Management split is Class A's cut of that above-hurdle profit. At 0.25, Class A (the manager) takes 25% of the excess and Class B keeps the other 75% on top of its hurdle. If the fund returns less than the hurdle, all of the gain goes to Class B and Class A takes nothing; in a losing year the loss is shared so Class A absorbs a portion rather than charging a fee.
